# licscore

用于自有 ThinkPHP 项目分发部署的授权扩展包。

## 说明
local 模式适合客户本地或内网部署，首次注册后通过本地授权文件校验。

online 模式适合需要云端实时管控的项目，网络异常时可能影响系统访问。

## 功能

- 初始化部署时引导站点注册
- 获取并保存授权文件
- 根据机器码绑定授权，防止授权文件复制到其他服务器使用
- 支持本地授权校验和远程授权校验

## 配置

配置文件：`application/extra/cloud.php`

```php
<?php

return [
    'url' => '',      // 授权服务地址
    'version' => '',  // 当前项目版本
    'type' => '',     // 项目标识
    'env' => 'local', // local 本地校验，online 远程校验
];
```
